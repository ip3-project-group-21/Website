Population Calculator

On this page the user will be able to get the idea of what each country's population size by seeing it in infrared format. By using this colour scheme, the user can get an idea on what a countries population size without knowing the number. If a user wants to know the exact number for the population size, they just have to hover over a country with their mouse.

For a more in-depth tutorial on how to fully take advantage of this page please follow the link here to our population tutorial page.
